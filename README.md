# 四十九蓍 (Forty-Nine)

一个交互式的数字蓍草占卜应用，模拟《易经》传统大衍筮法的完整流程。

## 演示

以 49 根蓍草起始，去一为太极，经十八变生成六爻，最终得出六十四卦之一。

## 技术栈

- React 19 + TypeScript
- Framer Motion（动画）
- Vite（构建工具）
- CSS Modules（样式）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 占卜流程

1. **分而为二（象两）** — 将蓍草随机分为左右两堆
2. **挂一** — 从右堆取出一根
3. **揲之以四（揲四）** — 左右两堆分别以四为一组计数
4. **归奇聚拢** — 收集余数，重新聚拢

重复以上步骤三次为一变，六变得一爻，十八变成一卦。根据六爻结果查得本卦，若有动爻则同时得出变卦。
